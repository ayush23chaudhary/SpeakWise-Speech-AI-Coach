## Metrics Calculation Details

### 1. Clarity Score
```
Input:  Word confidence values from Speech-to-Text
Logic:  Average of all word confidence scores
Output: 0-100 (higher = clearer pronunciation)

Example:
Words: ["hello"(0.95), "world"(0.92), "today"(0.89)]
Clarity = (0.95 + 0.92 + 0.89) / 3 * 100 = 92
```

### 2. Pace Analysis
```
Input:  Word start/end timestamps
Logic:  total_words / duration_minutes
Output: WPM + Status

Example:
30 words in 12 seconds = 150 WPM = "Good"

Ranges:
- Too Slow: < 130 WPM
- Good: 130-170 WPM  
- Too Fast: > 170 WPM
```

### 3. Fluency Score
```
Input:  Word timestamps + filler word count
Logic:  100 - (long_pauses * 4) - (filler_words * 2)
Output: 0-100 (higher = smoother flow)

Example:
2 long pauses (>1.5s) + 3 filler words
Fluency = 100 - (2*4) - (3*2) = 86
```

### 4. Filler Words
```
Input:  Transcript text
Logic:  Count occurrences of common fillers
Output: Map of word â†’ count

Detected: um, uh, like, you know, so, right, 
          actually, basically, i mean

Example:
"um, hello, like, I want to, um, speak"
â†’ { um: 2, like: 1 }
```

### 5. Confidence Score
```
Input:  Clarity, fluency, pace scores
Logic:  Weighted composite
Output: 0-100

Formula:
Confidence = (Clarity * 0.4) + (Fluency * 0.4) + (Pace * 0.2)

Example:
Clarity: 85, Fluency: 80, Pace: 90
Confidence = (85*0.4) + (80*0.4) + (90*0.2) = 84
```

### 6. Overall Score
```
Input:  All individual metric scores
Logic:  Weighted average
Output: 0-100

Formula:
Overall = (Clarity * 0.3) + (Fluency * 0.3) + 
          (Pace * 0.2) + (Confidence * 0.2)

Weights:
- Clarity: 30%
- Fluency: 30%
- Pace: 20%
- Confidence: 20%
```
